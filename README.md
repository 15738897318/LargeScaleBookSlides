# Slides for [Large Scale Convex Optimization via Monotone Operator](https://large-scale-book.mathopt.com/)
Authors: [Ernest K. Ryu](http://www.math.snu.ac.kr/~ernestryu/) and [Wotao Yin](https://wotaoyin.mathopt.com/)
